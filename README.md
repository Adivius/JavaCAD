# JavaCAD

It's a simple CAD program written in Java with my own graphics library for a school assignment.
With JavaCAD you can draw simple blueprints in 2D.

## Usage

    - Requires Java 8+
    - Requires Graphite 1.2.3+
    - Run Main.java without any args
